#### 编码测试输出结果示意

字符串[淘！我喜欢！]

解码字符集：ISO-8859-1

解码结果：

十进制表示：

63 63 63 63 63 63 

十六进制表示：

3f 3f 3f 3f 3f 3f 

编码字符集：ISO-8859-1

编码结果：??????

原因：ISO-8859-1不支持中文字符集，在遇到不在码值范围内的字符时同一用十六进制3f表示十进制为63，在ASII表中为?(英文问号)，注意，示例字符串中的！为中文感叹号。
